# /docs/plans - Implementation Roadmaps

This folder contains phase-based implementation plans.

---

## How to Use

1. **At project start:** Create MVP_PHASE1.md
2. **Before each phase:** Create next phase plan
3. **During development:** Reference current phase
4. **After phase complete:** Mark done, move to next

---

- Main user workflows
- Admin functionality
- Integrations

### Phase 3: Polish
- UX improvements
- Performance
- Testing

### Phase 4: Launch
- Final testing
- Documentation
- Client training
- Go live
